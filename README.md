# 🏥 Hospital SQL Data Analysis Project

## 📌 Project Overview

This project performs hospital data analysis using PostgreSQL. It answers business-related questions regarding patients, doctors, departments, medical expenses, and hospital stays.

## 📂 Files Included

### 📊 Dataset

* `Hospital_Data.xlsx`

### 📝 SQL Script

* `SQL1.sql` – Contains both the business questions and their SQL solutions.

### 📄 Documentation

* `SQLHOSPITALAnsAndQuery.pdf` – PDF containing the questions and answers.

## 🛠 Tools Used

* PostgreSQL
* SQL
* Microsoft Excel

## 📚 SQL Concepts Covered

* SELECT
* SUM()
* AVG()
* COUNT()
* GROUP BY
* ORDER BY
* LIMIT
* Date Arithmetic
* EXTRACT()

## 📈 Business Questions Solved

1. Total Number of Patients
2. Average Number of Doctors per Hospital
3. Top 3 Departments with Highest Number of Patients
4. Hospital with Maximum Medical Expenses
5. Average Medical Expenses
6. Longest Hospital Stay
7. Total Patients Treated Per City
8. Average Length of Stay Per Department
9. Department with Lowest Number of Patients
10. Monthly Medical Expenses Report

## 📄 Detailed Solutions

Detailed questions and SQL answers are available in:

* `SQL1.sql`
